# NGO Task & Staff Management System

## Overview
A simple full stack web application designed to help NGOs manage staff tasks, assignments, and progress tracking.

## Features
- User login (Admin & Staff)
- Create, assign, update, and delete tasks
- Task status tracking
- Simple dashboard overview
- RESTful API with Express.js
- Full CRUD operations (Create, Read,  Update, Delete)
- ES Module–based Node.js backend
- File-based persistence (JSON)
- Frontend–backend integration using Fetch API


## Tech Stack
- HTML, CSS, JavaScript
- Node.js (Backend)
- JSON / SQLite (Database)

## Purpose
This project demonstrates CRUD operations, backend logic, and real-world organizational workflows.
